# T-Mobile 셀룰러 커버리지 최적화 보고서
**대상 지역:** 캘리포니아주 로스앤젤레스 카운티  
**가용 예산:** $750,000  

## 1. 요약 (Executive Summary)
T-Mobile은 로스앤젤레스 카운티 내의 셀룰러 네트워크 커버리지를 극대화하기 위해 75만 달러의 자본 개선 예산을 할당했습니다. 이 보고서의 목적은 제안된 3가지 개발 시나리오 중 가장 효율적인 예산 사용 방안을 찾는 것입니다. 1각초(1-arcsecond) 고해상도 SRTM DEM 데이터를 활용한 고급 GIS 가시권(Viewshed) 분석 결과, **시나리오 C (새 타워 3개 건설)** 가 가장 높은 커버리지 면적 증가(현재 상태 대비 4.23% 순증가)를 가져오므로 이 방안을 권장합니다.

## 2. 방법론 (Methodology)
정확한 계산을 위해 다음과 같은 공간 분석 절차를 적용했습니다:
* **데이터 소스:** T-Mobile 타워 위치(`TMOSites.csv`), 로스앤젤레스 카운티 경계(`lacounty.shp`), USGS Earth Explorer에서 다운로드한 5개의 1각초 SRTM Void Filled DEM 타일.
* **데이터 준비:** 5개의 DEM 타일을 단일한 매끄러운 고도 래스터로 모자이크(이어붙이기)했습니다. 모든 지리 데이터(타워, DEM, 카운티 경계)는 정확한 미터 단위 면적 계산을 위해 **NAD 1983 UTM Zone 11N (EPSG:26911)** 좌표계로 투영 변환되었습니다.
* **가시권 매개변수:** 기존 구조물 높이를 기반으로 한 관찰자(타워) 높이, 1.5m의 수신자(사용자 휴대전화) 높이, 그리고 이상적인 대기 조건하에서 25,000미터(25km)의 기본 가시선 전송 반경을 가정하여 분석을 수행했습니다.
* **커버리지 계산:** 도출된 가시권은 로스앤젤레스 카운티의 정확한 경계에 맞춰 마스크(잘라내기) 처리되었습니다. 커버리지 면적은 전파가 도달하는(가시적인) 30m x 30m 픽셀의 면적을 모두 합산한 뒤 카운티의 총 토지 면적으로 나누어 계산했습니다.

## 3. 시나리오 분석 및 결과 (Scenario Analysis and Results)
아래 표는 현재 상태와 제안된 세 가지 시나리오 전반에 걸친 커버리지 면적의 변화를 보여줍니다:

| 개발 시나리오 | 커버리지 면적 (㎡) | LA 카운티 커버리지 비율 | 현재 상태 대비 순증가 |
| :--- | :--- | :--- | :--- |
| **현재 상태 (기준)** | 4,929,452,100 ㎡ | 40.08% | - |
| **시나리오 A (+10m 높이)** | 5,008,313,700 ㎡ | 40.72% | + 0.64% |
| **시나리오 B (+5km 범위)** | 5,237,206,200 ㎡ | 42.58% | + 2.50% |
| **시나리오 C (새 타워 3개 건설)** | 5,449,495,500 ㎡ | 44.31% | **+ 4.23%** |

*(결과에 대한 시각적인 비교는 `Coverage_Comparison_Chart.png`를 참조하십시오.)*

### 3.1 현재 상태 (Status Quo)
현재 T-Mobile의 네트워크는 로스앤젤레스 카운티 총 토지 면적의 약 **40.08%**를 커버합니다. 커버리지는 인구 밀도가 높은 도심 분지에 집중되어 있는 반면, 산악 지역과 북부 외곽 지역은 지형적 장애물(가시선 차단)로 인해 통신 서비스가 상당히 부족한 상태입니다.

### 3.2 시나리오 A: 기존 타워 높이 10m 증가
모든 기존 타워의 높이를 10미터 높여도 전체 커버리지는 **40.72%**(0.64% 증가)로 아주 미미한 개선에 그칩니다. 기존 타워 대부분이 비교적 평탄한 도심 지역에 위치하고 있기 때문에, 높이를 추가하더라도 신호가 서비스 소외 계곡에 도달하는 것을 막는 거대한 지형적 장벽(예: 산 가브리엘 산맥)을 효과적으로 극복하지 못합니다.

### 3.3 시나리오 B: 전송 범위 5km 확장
모든 기존 타워의 범위를 25km에서 30km로 확장하면 적당한 수준의 개선(**42.58%** 전체 커버리지)이 나타납니다. 기존 타워가 계곡 쪽으로 신호를 조금 더 멀리 밀어낼 수는 있지만, 여전히 거대한 산맥에 막혀 신호가 카운티의 북부 깊숙한 곳까지 도달하지 못합니다.

### 3.4 시나리오 C: 통신 소외 지역에 새 타워 3개 건설
가장 큰 "데드존(어떤 기존 타워의 25km 반경에도 속하지 않는 지역)"에 완전히 새로운 타워 3개를 전략적으로 배치함으로써, 네트워크는 기존의 지리적 장벽을 완전히 극복합니다. 이 시나리오는 투자 수익을 극대화하여 총 커버리지를 **44.31%**(4.23%의 획기적인 순증가)로 끌어올립니다.

## 4. 최종 권고안 (Final Recommendation)
공간 분석 결과에 따르면, 로스앤젤레스 카운티에서 T-Mobile의 셀룰러 커버리지를 확장하는 데 가장 효과적인 접근 방식은 **시나리오 C (새 타워 3개 건설)** 임이 명백합니다. 기존 인프라의 범위(시나리오 B)나 높이(시나리오 A)를 개선하는 것은 카운티의 산악 지형으로 인해 수익 체감 현상을 겪는 반면, 서비스가 제공되지 않는 지역에 새로운 원점을 구축하면 이러한 물리적 장애물을 성공적으로 우회할 수 있습니다. 따라서 750,000달러의 예산은 이 세 개의 새로운 셀룰러 타워 건설에 할당할 것을 권장합니다.
