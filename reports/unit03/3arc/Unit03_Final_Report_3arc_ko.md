# T-Mobile 셀룰러 커버리지 최적화 보고서 (3각초 해상도 분석)
**대상 지역:** 캘리포니아주 로스앤젤레스 카운티  
**가용 예산:** $750,000  
**해상도:** 3각초 (픽셀당 약 90m x 90m)

## 1. 요약 (Executive Summary)
T-Mobile은 로스앤젤레스 카운티 내의 셀룰러 네트워크 커버리지를 극대화하기 위해 75만 달러의 자본 개선 예산을 할당했습니다. 이 보고서의 목적은 제안된 3가지 개발 시나리오 중 가장 효율적인 예산 사용 방안을 찾는 것입니다. 1각초 데이터보다 계산이 빠르고 가벼운 3각초(3-arcsecond) SRTM DEM 데이터를 활용한 고급 GIS 가시권(Viewshed) 분석 결과, **시나리오 C (새 타워 3개 건설)** 가 가장 적합한 것으로 명백하게 권장됩니다. 이 방안은 가장 높은 커버리지 면적 증가(현재 상태 대비 4.96% 순증가)를 가져오며, 지리적인 신규 배치가 장애물에 가려진 기존 인프라를 업그레이드하는 것보다 훨씬 효과적임을 증명합니다.

## 2. 방법론 (Methodology)
처리 속도를 최적화하면서도 정확한 계산을 보장하기 위해 다음과 같은 공간 분석 절차를 적용했습니다:
* **데이터 소스:** T-Mobile 타워 위치(`TMOSites.csv`), 로스앤젤레스 카운티 경계(`lacounty.shp`), USGS Earth Explorer에서 다운로드한 5개의 3각초 SRTM DEM 타일.
* **데이터 준비:** 5개의 DEM 타일을 단일한 매끄러운 고도 래스터로 모자이크(이어붙이기)했습니다. 모든 지리 데이터(타워, DEM, 카운티 경계)는 정확한 미터 단위 면적 계산을 위해 **NAD 1983 UTM Zone 11N (EPSG:26911)** 좌표계로 투영 변환되었습니다.
* **가시권 매개변수:** 기존 구조물 높이를 기반으로 한 관찰자(타워) 높이, 1.5m의 수신자(사용자 휴대전화) 높이, 그리고 이상적인 대기 조건하에서 25,000미터(25km)의 기본 가시선 전송 반경을 가정하여 분석을 수행했습니다.
* **커버리지 계산:** 도출된 가시권은 로스앤젤레스 카운티의 정확한 경계에 맞춰 마스크(잘라내기) 처리되었습니다. 커버리지 면적은 전파가 도달하는(가시적인) 90m x 90m 픽셀의 면적을 모두 합산한 뒤 카운티의 총 토지 면적으로 나누어 계산했습니다.

*(참고: 3각초 데이터는 1각초 데이터와 비교할 때 커버리지를 약간 과대평가하는 경향이 있습니다. 더 거친 90m 픽셀은 고해상도인 30m 모델에서는 신호를 차단했을 법한 작은 지형적 장애물들을 수학적으로 "매끄럽게" 덮고 넘어가기 때문입니다.)*

## 3. 시나리오 분석 및 결과 (Scenario Analysis and Results)
아래 표는 3각초 고도 모델을 사용하여 계산한, 현재 상태와 제안된 세 가지 시나리오 전반에 걸친 커버리지 면적의 변화를 보여줍니다:

| 개발 시나리오 | 커버리지 면적 (㎡) | LA 카운티 커버리지 비율 | 현재 상태 대비 순증가 |
| :--- | :--- | :--- | :--- |
| **현재 상태 (기준)** | 5,357,234,700 ㎡ | 43.56% | - |
| **시나리오 A (+10m 높이)** | 5,408,896,500 ㎡ | 43.98% | + 0.42% |
| **시나리오 B (+5km 범위)** | 5,693,741,100 ㎡ | 46.29% | + 2.73% |
| **시나리오 C (새 타워 3개 건설)** | 5,968,120,500 ㎡ | 48.52% | **+ 4.96%** |

*(결과에 대한 시각적인 비교는 `Coverage_Comparison_Chart_3arc.png`를 참조하십시오.)*

### 3.1 현재 상태 (Status Quo)
현재 T-Mobile의 네트워크는 로스앤젤레스 카운티 총 토지 면적의 약 **43.56%**를 커버합니다. 커버리지는 인구 밀도가 높은 도심 분지에 집중되어 있는 반면, 산악 지역과 북부 외곽 지역은 지형적 장애물(가시선 차단)로 인해 통신 서비스가 상당히 부족한 상태입니다.

### 3.2 시나리오 A: 기존 타워 높이 10m 증가
모든 기존 타워의 높이를 10미터 높여도 전체 커버리지는 **43.98%**(0.42%의 극히 적은 증가)로 거의 무의미한 개선을 보입니다. 기존 타워 대부분이 비교적 평탄한 도심 지역에 위치하고 있기 때문에, 10미터의 높이를 추가하더라도 신호가 서비스 소외 계곡에 도달하는 것을 막는 거대한 지형적 장벽(예: 산 가브리엘 산맥)을 넘을 수 있는 각도를 제공하지 못합니다.

### 3.3 시나리오 B: 전송 범위 5km 확장
모든 기존 타워의 범위를 25km에서 30km로 확장하면 적당한 수준의 개선(**46.29%** 전체 커버리지)이 나타납니다. 기존 타워가 계곡 쪽과 해안 평야를 따라 신호를 조금 더 멀리 밀어낼 수는 있지만, 여전히 주요 산맥에 의해 신호가 근본적으로 차단되어 카운티 북부 깊숙한 곳까지 도달하지 못합니다.

### 3.4 시나리오 C: 통신 소외 지역에 새 타워 3개 건설
가장 큰 "데드존(어떤 기존 타워의 25km 반경에도 속하지 않는 지역)"에 완전히 새로운 타워 3개를 전략적으로 배치함으로써, 네트워크는 기존의 지리적 장벽을 완전히 극복합니다. 이 시나리오는 투자 수익을 극대화하여 총 커버리지를 **48.52%**(4.96%의 상당한 순증가)로 끌어올립니다.

## 4. 최종 권고안 (Final Recommendation)
공간 분석 결과에 따르면, 로스앤젤레스 카운티에서 T-Mobile의 셀룰러 커버리지를 확장하는 데 가장 효과적인 접근 방식은 **시나리오 C (새 타워 3개 건설)** 임이 명백합니다. 

1각초 데이터를 사용하든 3각초 데이터를 사용하든 지리적인 핵심 결론은 동일합니다: 카운티의 험준한 산악 지형으로 인해, 기존 인프라의 범위(시나리오 B)나 높이(시나리오 A)를 개선하는 것은 극심한 수익 체감 현상을 겪게 됩니다. 반면 서비스가 제공되지 않는 지역에 새로운 전파 발생 지점을 구축하면 이러한 물리적 장애물을 성공적으로 우회할 수 있습니다. 따라서 750,000달러의 예산은 이 세 개의 새로운 셀룰러 타워 건설에 전액 할당할 것을 강력히 권장합니다.
