Unit 06: Darién Gap 비용거리 분석

1. 과제 정의

  • 해결 과제: Pan-American Highway의 단절 구간인 Darién Gap을 통과하는 가상의 도로 후보를 least-cost path로 탐색합니다.
  • 출발지: Yaviza, Panama. 도착지: Apartadó, Colombia와 Chigorodó, Colombia 두 지점입니다.
  • 비용 기준: terrain slope가 커질수록 비용이 비선형적으로 증가하고, 해수면 이하 또는 대형 수역으로 볼 수 있는 셀에는 큰 penalty를 부여했습니다.
  • 제출 요구: 두 least-cost path 지도, 두 elevation profile, 하나 이상의 3D terrain view, 분석 결과와 다음 반복에서 필요한 추가 변수 설명입니다.

2. Python 재현 산출물

142.5 km Yaviza to Chigorodó
188 m 두 경로의 최대 표고
10,569 OSM road context ways
PDF 보고서 열기 경로 요약 CSV 데이터 출처 CSV 경로 좌표 CSV
🖱️

ArcGIS Pro 구현

Spatial Analyst 기반 비용거리 분석

STEP 01

DEM 준비와 slope 산출

SRTM Void-Filled DEM을 Darien Gap 분석 범위로 mosaic/clip하고, Slope 도구로 경사도를 계산합니다.

STEP 02

비용면 설계

Reclassify 또는 Raster Calculator를 사용해 slope 비용을 비선형으로 키우고, 대형 수역 또는 해수면 이하 셀에는 높은 비용을 부여합니다.

STEP 03

Cost Distance / Cost Path

Yaviza를 source로 Cost Distance와 Back Link를 계산한 뒤, Apartadó와 Chigorodó를 destination으로 각각 Cost Path를 생성합니다.

STEP 04

지도와 3D 결과 검토

국경, 도로망, 후보 경로를 한 지도에 배치하고, Profile Graph와 3D Scene으로 표고 변화를 검토합니다.

🐍

Python 구현

DEM 샘플링, Dijkstra pathfinding, PDF 자동 생성

DATA

공개 데이터만 사용

Mapzen/AWS Terrain Tiles DEM, DIVA-GIS 국가 경계, OpenStreetMap 도로를 다운로드하고 출처 manifest를 남겼습니다.

MODEL

비용 공식

기본 비용은 1 + (slope / 8)^2.2이며, DEM이 2m 이하인 셀은 large-water proxy로 보고 180을 추가했습니다.

OUTPUT

제출 파일 자동 생성

지도, 프로파일, 3D view, PDF 보고서, 경로 요약 CSV, 경로 좌표 CSV, 출처 manifest가 한 번에 생성됩니다.
